# Hospital Management System (Django)

A full-stack Hospital Management System built using Django with role-based access for Doctors and Patients.

## Features
- Doctor & Patient Login
- Appointment Booking
- Doctor Availability
- Google OAuth Login
- Clean UI
- PostgreSQL Database

## Tech Stack
- Django
- PostgreSQL
- HTML, CSS, Bootstrap
- Django-Allauth

## Run Project
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
